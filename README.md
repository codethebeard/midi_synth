# midi_trellis
An Arduino sketch for a Trellis on a Arduino Pro Micro as a MIDI controller. The Arduino Pro Micro communicates over I2C

## Pin Diagram

| Pin | Trellis       |
| ------------- |:-------------:|
| VCC      | 5V |
| GND      | GND      |
| 2A | INT      |
| 2 | SDA      |
| 3 | SCL |
